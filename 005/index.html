  • 2. べびたっぴ

Q6. Windows 7

Windows 7のサポート終了日となる2020年1月14日まで、残すところあと約1ヶ月となりましたが、初期の開発コードネームは何？

  • 1. Cheetah

  • 2. Blackcomb

  • 3. Catalina

マイクロソフトが2009年にリリースした、Windowsシリーズに属するパーソナルコンピュータ用のオペレーティングシステム (OS) である。初期の開発コードネームは「Blackcomb」であったが、のちに「Vienna」となり、さらに「Windows 7」に改められた。本来次期クライアント用 Windows の社内開発コードネームだったものが、そのまま製品版の名称として採用された。

Q7. アニメ流行語

ガジェット通信 アニメ流行語大賞2019 において、金賞になったアニメ流行語は何？

  • 1. ゴマすりクソバード

  • 2. シャミ子が悪いんだよ

  • 3. イキリ鯖太郎

金賞は放送前から何かと話題になった『けものフレンズ2』から、第7話「すぴーどのむこう」に登場したG・ロードランナーのネットでの愛称、「ゴマすりクソバード」。発祥はTwitter、ニコニコ静画に投稿された4コマ漫画からですが、この呼名を略した「ゴマクソちゃん」「ゴマちゃん」など、もはや元の名前の原型のない形で親しまれ、ファンの作品に対する様々な想いにより支持を集めるキャラクターとなっています。
金賞：ゴマすりクソバード（けものフレンズ2）
銀賞：シャミ子が悪いんだよ（まちカドまぞく）
銅賞：イキリ鯖太郎（Fate/Grand Order -絶対魔獣戦線バビロニア-）
